## Overview

本库定义《咫尺》中所有可复用的交互模式。每个模式包含行为规范、输入映射、反馈机制和无障碍要求。新屏幕设计时应优先引用已有模式，保持一致性。

**输入方法**: Keyboard/Mouse (主), Touch (全支持), Gamepad (部分)
**目标平台**: Web (HTML5), 桌面 + 移动浏览器

## Patterns

### P01: 印章按钮 (Seal Button)

**类别**: Navigation
**使用场景**: 主菜单、暂停菜单、确认对话框、功法夺取确认

**描述**: 方印形状按钮，灵感来自中国传统印章。墨色边框 + 纸白底 + 楷体文字。hover 时朱砂红描边，press 时下沉效果。

**规范**:
- 形状: 圆角矩形 (2px corner radius)
- 尺寸: 桌面 40h, 触屏 64h, 最小宽度 120px
- 边框: 墨黑 1px
- 背景: 宣纸白 `#F0E8D8`
- 文字: 楷体, 墨黑
- 状态:
  - Normal: 墨黑边框
  - Hover: 朱砂红边框 + 15% 朱砂红覆盖 (80ms ease-out)
  - Press: 1px 下沉 + 40% 朱砂红覆盖 (60ms linear)
  - Disabled: 50% 灰墨, 无 hover 响应

**输入映射**:
| 平台 | 触发方式 |
|------|----------|
| 桌面 | 鼠标点击 / Enter (焦点时) |
| 触屏 | 点击 |
| 手柄 | A 按钮 (焦点时) |

**无障碍**:
- 最小触控目标: 44×44px (W3C)
- 焦点指示器: 朱砂红外框 2px
- 键盘导航: Tab 移动焦点, Enter 确认

**When to Use**: 所有确认性操作
**When NOT to Use**: 连续性操作 (如拖动)

### P02: 径向冷却指示器 (Radial Cooldown)

**类别**: Feedback
**使用场景**: 技能槽冷却显示

**描述**: 朱砂红从 12 点方向顺时针擦除，显示剩余冷却进度。

**规范**:
- 形状: 方印内圆形擦除
- 颜色: 朱砂红 `#B8403A` (冷却中)
- 动画: 顺时针从 12 点, angle = elapsed/total × 360°
- 完成时: 200ms 脉冲 (朱砂环外扩 4px + 字体闪白)

**输入映射**: 无交互 (纯显示)

**无障碍**:
- 冷却完成时有音效提示
- 色盲安全: 形状变化 (脉冲) 不依赖颜色

### P03: 灵气条 (Spirit Bar)

**类别**: Data Display
**使用场景**: HUD 顶部灵气进度

**描述**: 水平线，墨色填充由左至右晕开 (卷轴展开效果)。突破节点标记。

**规范**:
- 尺寸: 3px 高, 屏幕宽度 60% (0.20-0.80)
- 填充: 墨色渐变晕开
- 节点: 4-5 个标记 (对应 R1-R5)
  - 已到达: 朱砂红
  - 未到达: 淡墨轮廓
- 透明度: 常驻 α=0.5, 战斗中 α=1.0
- 填充满时: 朱砂红闪光 200ms

**输入映射**: 无交互 (纯显示)

**无障碍**:
- 数值变化时有音效 (上升音)
- 色盲安全: 填充进度不依赖颜色 (位置变化即可读)

### P04: BOSS 血条 (Boss HP Bar)

**类别**: Data Display
**使用场景**: BOSS 战顶部居中

**描述**: 水平线，朱砂红填充。仅 BOSS 战时显示。

**规范**:
- 位置: 顶部居中 (0.50, 0.04)
- 尺寸: 屏幕宽度 30%, 4px 高
- 填充: 朱砂红 `#B8403A`
- 双层反馈: 前层即时 (朱砂红), 后层 200ms 延迟 (白色)
- BOSS 名称: 血条上方, 朱砂印章

**输入映射**: 无交互 (纯显示)

**无障碍**:
- BOSS 死亡时有音效
- 色盲安全: 填充位置变化即可读

### P05: 虚拟摇杆 (Virtual Joystick)

**类别**: Input
**使用场景**: 触屏移动控制

**描述**: 半透明圆形摇杆，触屏拖动控制角色移动。

**规范**:
- 位置: 底部左侧 (0.04, 0.94)
- 外圈: 80px 直径, 30% 透明度
- 内圈: 30px 直径, 跟随手指
- 死区: 10% (inner dead zone)
- 响应曲线: 1.5 次方 (轻度非线性)

**输入映射**:
| 平台 | 触发方式 |
|------|----------|
| 触屏 | 拖动 |
| 桌面 | 不显示 (WASD/方向键) |
| 手柄 | 左摇杆 |

**无障碍**:
- 触控目标: 80px + 16px 超调
- 视觉反馈: 按下时朱砂红闪光

### P06: 墨色横扫转场 (Ink Sweep Transition)

**类别**: Transition
**使用场景**: 场景切换 (菜单↔战斗)

**描述**: 墨色从右到左横扫覆盖，内容切换后墨色消散。

**规范**:
- 覆盖阶段: 300ms, 墨色从右到左
- 消散阶段: 200ms, 墨色从左到右
- 颜色: 浓墨 `#1A1D22`

**输入映射**: 转场期间禁止输入

**无障碍**:
- 可通过 reduced-motion 设置跳过动画

### P07: 朱砂脉冲反馈 (Cinnabar Pulse)

**类别**: Feedback
**使用场景**: 技能就绪、灵气吸收、突破完成

**描述**: 朱砂红色向外扩张然后消散的短促脉冲。

**规范**:
- 持续: 200ms
- 动画: 0-100ms 环外扩 4px + fade, 50-150ms 字体闪白
- 颜色: 朱砂红 `#B8403A`

**输入映射**: 无交互 (纯反馈)

**无障碍**:
- 同时有音效提示
- 色盲安全: 形状变化 (扩张) 不依赖颜色

---

### P08: 弹窗/模态 (Modal)

**类别**: Modal
**使用场景**: 暂停菜单、设置、功法选择

**描述**: 纸边矩形模态，浮于游戏上方。

**规范**:
- 背景: 30% 暗化 + 去饱和
- 容器: 矩形 + 四角微飞白磨损
- 尺寸: 屏幕 60% 宽度, 自适应高度
- 动画: 进入 200ms scale 95%→100% + fade, 退出 150ms fade

**输入映射**:
| 平台 | 关闭方式 |
|------|----------|
| 桌面 | ESC / 点击背景 / 关闭按钮 |
| 触屏 | 返回手势 / 关闭按钮 |
| 手柄 | B 按钮 |

**无障碍**:
- 焦点陷阱: Tab 循环在模态内
- ESC 可关闭
- 屏幕阅读器: aria-modal="true"

---

### P09: 卷轴滚动 (Scroll View)

**类别**: Navigation
**使用场景**: 功法图鉴、设置列表

**描述**: 横向或纵向滚动容器，水墨风格。

**规范**:
- 滚动条: 细线 (2px), 墨色, 30% 透明度
- 边缘: 淡入淡出 (暗示更多内容)
- 动画: 惯性滚动, 300ms 减速

**输入映射**:
| 平台 | 滚动方式 |
|------|----------|
| 桌面 | 鼠标滚轮 / 拖动 |
| 触屏 | 滑动 |
| 手柄 | 摇杆 |

**无障碍**:
- 键盘: 方向键滚动
- 手柄: 右摇杆滚动

---

### P10: 境界印章 (Realm Seal)

**类别**: Data Display
**使用场景**: HUD 右上角境界标识

**描述**: 朱砂印章，显示当前境界名称。

**规范**:
- 位置: 右上角 (0.96, 0.06)
- 形状: 方印
- 颜色: 朱砂红底 + 白字
- 突破时: 短暂放大 + 朱砂闪光

**输入映射**: 无交互 (纯显示)

**无障碍**:
- 境界变化时有音效
- 色盲安全: 形状 (方印) 不依赖颜色
